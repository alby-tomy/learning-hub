About the programme

A design education built for engineers, not designers-in-waiting

UI/UX Path exists because design literacy shouldn't be reserved for people with "Designer" in their title. Over 2 immersive weeks, you'll move through the same process working design teams use — from empathizing with users to handing off a polished, accessible interface to engineering.

What you'll walk away with

Three ways of working, reinforced daily

Think like a designer

Five-stage Design Thinking, research methods, and information architecture — the reasoning behind every screen.

Build like a practitioner

Wireframes, design systems, and prototypes built in Figma, the same tool used in production teams.

Ship like a professional

Usability testing, responsive handoff, and a portfolio case study ready to show a hiring manager.
